# 떡밥

> 생성기가 매 화 갱신([열림]→[회수]). 열린 떡밥이 소리없이 사라지면 연속성 체크가 잡는다.

## 열림
- [중심] **[[발루아 백작]] 누명의 첫 씨앗 = '4황자 밀약' 소문** — 전생에 아버지를 반역으로 몬 누명의 시작점. 21화에서 충정 재프레이밍으로 1차 무력화. 단 모함 문서의 손이 [[황후]] 의전관 부봉임이 드러나 위기 격상.
- [중심] **모함의 진짜 손 = [[황후]] (사냥개 아닌 주인)** — 발루아를 친 첫 투서에 황후 직속 의전관 부봉. [[아드리안]] 매 인장 아님. 황후-아드리안 분리 쐐기 후속 정황이자, 세실리아만 쥔 정보 우위(봉서 1장 보유). 황후 라인의 직접 공격 예고.
- [중심] **[[황후]]의 혼사 하사 — 시계가 폐하 손으로** — 식 시점·'그릇' 판단 공개 위임으로 무한 보류화. 혼사 자체는 닫히지 않음(승낙도 거절도 아님). 발루아를 그늘에 묶는 수와 병행.
- [중심] **[[아드리안]]의 집착 — 보호 미끼 반격** — 시계를 뺏긴 뒤 21화 첫 반격: 소문 위기에 '공작가 이름으로 지켜 주겠다, 식을 앞당기자'는 devotion-as-protection으로 시계 되쥐기 시도(차단됨). 다음 수 미정.
- [중심] **회피선 붕괴 = 2막 진입** — 세실리아는 공인 '공작가 약혼 후보'. 공개 대면이 상시 전장.
- [중심] **발루아 무계파 위치 상실** — 21화 충신 프레임으로 반걸음 복원. 잔여 손실·황후 그늘 묶임은 미수습.
- [중심] **[[세실리아]]의 슬립 + 회귀 의심** — [[아드리안]] 보유, 미해소. 이번 화는 잠복(직격 없음).
- [중심] **[[로잘린]] = [[아드리안]] 직통 채널 (같은 둥지)** — 집 안 상존. 21화 서재 봉서를 가로채려다 세실리아에게 선수 빼앗겨 빈손(빼앗긴 아이의 표정 1회·대사 없음). 핀 출처 흐림 유지. 동기=결핍·허기(행동·표정만, 대사 금지).
- [중심] **[[황후]]-[[아드리안]] 같은 그늘 라인** — 분리 쐐기 진행. 모함이 황후 손에서 나옴 = 사냥개와 주인이 따로 움직이는 정황. 둘을 가를 틈 확대.
- [중심] **[[디미트리]] '쥐지 않은 손' — 비공개 통로 재개** — 공개 후퇴 상태 유지하나 카심 경유로 '등은 봐준다'+소문 출처('위에서 내려왔다') 정보 제공. 물러선 자리의 온도가 가까운 어느 손보다 따뜻함 — 진짜 인연 대비 심화.
- [장기] **[[발루아 백작]] 지키기** — 곡물세 반려 유지+충신 프레임 1승, 그러나 황후발 직접 공작으로 위험 최고조.
- [장기] **[[로잘린]]의 배신** — 동기(허기) 누적, 봉서 가로채기 실패 후 다음 함정 대기.
- [장기] **회귀의 비밀** — 미해명 유지.
- [암시] **변경의 칼 / [[디미트리]] 라인** — 비공개 통로 재개로 라인 재가동 정황.
- [열림] **[[엘비라 마르샹]] 우군화** — 부서-자작가 채널 상존(이번 화 직접 등장 없음).
- [열림] **[[세실리아]]의 회귀 은폐** — 의심 상존.
- [열림] **[[발루아 백작 부인]](어머니)** — 곡물세 반려 가내 공유 미완 유지(이번 화 직접 등장 없음).
- [열림] **세실리아 vs '진짜 인연' 대비** — 곁을 비우면서도 어둠 속 등을 봐주는 [[디미트리]] 대 다정함으로 가두려는 [[아드리안]]. 21화 카심 쪽지 온도로 진전.

## 회수
- **아드리안 시간표 박탈(20화)** → 21화 보호 미끼 반격으로 되쥐기 시도→차단(닫지 않고 [중심] 승계).
- **발루아 무계파 상실(20화)** → 21화 충신 재프레이밍으로 반걸음 복원(잔여 손실·승계).
- **디미트리 공개 후퇴(20화)** → 21화 비공개 통로 재개로 형질 전환(승계).
- **[[황후]] 혼사 하사 '그릇' 보류(19화)→공개 위임(20화)** → 무한 보류 유지([중심] 승계).
- **[[황후]] 차회 호출(18화)** → 19~20화 처리, 21화 모함 주체로 황후 판 재가동.
- **17화 아드리안 적중의 정보 출처** → 19화 로잘린 질투발 실수로 소급 확정(유지).
- **[[리타]] 보호 / '환한 자리'** → 18화 완결(유지·관리).
- **세실리아 비밀 편지 봉쇄** → 15화 공개 자선 전환 완결(유지).
- **'살아서 갚으라' 값([[디미트리]] 라인)** → 16화 일부 호출·거절, 보류 유지.
- **운명의 무도회** → 9화 닫음(유지).
- **'벗삼아' 셋째 손** → 9화 폐하-황후 층위 정리(유지).
