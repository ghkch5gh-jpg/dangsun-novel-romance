# 떡밥

> 생성기가 매 화 갱신([열림]→[회수]). 열린 떡밥이 소리없이 사라지면 연속성 체크가 잡는다.

## 열림
- [중심] **[[아드리안]]의 집착 — 회귀 의심 단계** — 14화 ‘한 번 겪어 본 사람처럼’ 직격을, 15화에서 세실리아가 ‘데어 본 자의 조심성’으로 무마. 다만 *미해소* — 언제든 재점화 가능. 15화에선 변경 편지를 ‘디미트리 구애’로 **오독**(전지전능 첫 균열). 다음 수: 오독이 풀리거나, 의심을 키울 본인 접근.
- [중심] **무관심이라는 무기** — ‘멈추면 더 깊이 파고든다’ 원리 유지. 15화는 ‘맞서 막기’ 대신 ‘환한 데 펼쳐 무해화’로 변주 — 무관심의 진화형.
- [중심] **[[세실리아]]의 슬립 한 조각 + 회귀 의심** — 아드리안 보유 유지. 15화로 회귀 의심은 일시 무마됐으나 회귀 자체는 미노출.
- [중심] **[[로잘린]] = [[아드리안]] 직통 채널 (같은 둥지)** — 14화 직접 함정(‘변경’ 입으로 심기)이, 15화에서 세실리아가 장부를 공개로 펼치며 **무해한 패로 중화**됨(이번 수 빈손). 채널은 집 안에 상존. 핀 출처 흐림 유지.
- [중심] **[[황후]]의 판** — 빚 없이 빠진 발루아를 끌어들일 다음 손 남음. 15화 직접 등장 없음(아침 후속 미도래, 다음 화 강력 후보). 발루아-4황자 공개 결합을 황후가 어찌 읽을지 새 긴장.
- [장기] **[[발루아 백작]] 지키기** — 곡물세 반려 유지로 전생 죽음 통로 차단. 15화 등장 없음.
- [장기] **[[리타]] 지키기** — 표적 확정. 변경 도피 봉쇄 유지. 15화에서 4황자 공개 채널이 열리며 ‘전속을 빼돌리지 않고 지킬’ 새 토대 마련 — 구체화는 다음 화.
- [장기] **[[로잘린]]의 배신** — 직접 함정 단계. 15화는 빈손으로 돌아갔으나 다음 함정 예고.
- [장기] **회귀의 비밀** — 미해명 유지. 아드리안 의심은 일시 무마.
- [암시] **변경의 칼 / [[디미트리]] 라인 우군화** — 봉쇄된 비밀 편지 대신 **공개 자선 장부**로 통로 개설 성공. 디미트리 부서가 먼저 회신(자작가에도 같은 청). ‘살아서 갚으라’ 값의 공개 회수 착수 — 우군화 1보 실현.
- [열림] **[[엘비라 마르샹]] 우군화** — 무도회 호의가 차회 초대로 돌아옴. 부서 회신 전달·공개 서신 운반의 자발적 통로가 됨. 본인은 아직 자신이 무엇의 연결점인지 모름.
- [열림] **[[세실리아]]의 회귀 은폐** — 아드리안의 ‘조심성’ 수용 + ‘구애’ 오독으로 압박 일시 완화. 오독이 풀리는 순간이 다음 위기.
- [열림] **[[발루아 백작 부인]](어머니)** — 우군 자원 유지. 곡물세 반려 가내 공유는 여전히 미완.
- [열림] **세실리아의 첫 능동책(집 밖에 손 두기)** — 14화 비밀 편지 봉쇄를 15화에서 ‘공개 채널 전환’으로 **재가동·완결**. 다음은 이 채널의 확장과 후폭풍.

## 회수
- **세실리아의 첫 능동책 / 비밀 편지 봉쇄** → 15화 ‘공개 자선 장부’ 전환으로 재설계·완결(첫 완결된 1승). 변경 비밀 경로는 폐기, 공개 채널로 대체.
- **‘살아서 갚으라’ 값 ([[디미트리]] 라인)** → 15화 공개 장부 위에서 회수 착수(부서 회신). 실집행은 다음 화.
- **카심 ‘모십니다’ 어법 균열(6화)** → 15화 사교계 공개 우스개로 회수 — 발루아-4황자 연결의 무해한 위장으로 전용됨.
- **2~3화 ‘약혼 제안’ 후폭풍** → 15화 아드리안의 ‘디미트리 구애’ 오독으로 회수(질투 오독의 근거로 작동, 세실리아가 역이용).
- **리타 ‘잠시 빌려 가겠다’ 위기** → 13화 1차 해소, ‘약점 확보’ 전이(열림 유지).
- **세실리아의 역이용 전략(‘진실인 척 빚은 것’)** → 15화 ‘위장을 아예 진짜 공개 자선으로 굳혀 무해화’로 진화.
- **아드리안 매 인장 봉서/초/쪽지** → 10~11화 처리, 소매 보관 유지.
- **운명의 무도회** → 9화 닫음(유지).
- **마르샹 자작가를 끌어 올린 손 / ‘벗삼아’ 셋째 손** → 9화 폐하-황후 층위로 정리(유지).
