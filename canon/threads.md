# 떡밥

> 생성기가 매 화 갱신([열림]→[회수]). 열린 떡밥이 소리없이 사라지면 연속성 체크가 잡는다.

## 열림
- [중심] **[[황후]]의 혼사 하사** — 19화 차회에서 [[세실리아]]를 [[아드리안]]과 직접 맺어 주겠다 선언. 거절=황실 모독. 세실리아는 '그릇이 차기까지'로 보류(빚 없음). 거절하는 손을 거절할 수 없게 묶는 정략. **막경계(20화) 정면 후보.**
- [중심] **[[아드리안]]의 집착 — 매혹으로의 전환** — 19화 차회 직접 등장, 위협이 아닌 전생의 다정한 음색으로 세실리아의 통제를 처음 깸(흔들림 1회 유발). 보류 시간을 '천천히 지켜보겠다'며 사냥터로 가져감. 회귀 의심 자체는 미해소(이번 화는 의심 비유 없이 행동=집착으로).
- [중심] **무관심이라는 무기** — 서신 답장 0 유지. 단 차회 대면에선 직접 응수 불가피(별개 전장).
- [중심] **[[세실리아]]의 슬립 한 조각 + 회귀 의심** — [[아드리안]] 보유, 미해소. 18화 균열로 압박 일시 둔화 유지.
- [중심] **[[로잘린]] = [[아드리안]] 직통 채널 (같은 둥지)** — 집 안 상존. 19화 질투발 실수('장부'와 '이름'을 같은 순서로 부러워함)로 **17화 아드리안 적중의 정보 출처가 로잘린 채널임이 소급 확정**. 핀 출처 흐림 유지. 배신 동기=결핍·허기(반복 노출 절제).
- [중심] **[[황후]]-[[아드리안]] 같은 그늘 라인** — 19화 아드리안이 황후 차회에 거리낌 없이 듦으로 표면화. '벗삼아' 부봉의 손과 집착의 가지가 한 그늘. 둘을 가를 틈이 향후 과제.
- [장기] **[[발루아 백작]] 지키기** — 곡물세 반려 유지. 단 혼사로 묶이면 반려까지 무력화될 연쇄 위험.
- [장기] **[[로잘린]]의 배신** — 동기(허기) 누적. 다음 함정 예고.
- [장기] **회귀의 비밀** — 미해명 유지.
- [암시] **변경의 칼 / [[디미트리]] 라인 우군화** — 18화 장부실 동석 이후 19화 직접 등장 없음. 곡물세/황후 판 재요동 대기. '쥐지 않은 손' 온도(진짜 인연 대비) 보류.
- [열림] **[[엘비라 마르샹]] 우군화** — 19화 직접 등장 없음. 부서-자작가 결합 채널 가동 중.
- [열림] **[[세실리아]]의 회귀 은폐** — 의심 상존, 압박 일시 완화.
- [열림] **[[발루아 백작 부인]](어머니)** — 곡물세 반려 가내 공유 미완 유지.
- [열림] **세실리아 vs '진짜 인연' 대비** — 약점 안 쑤시는 [[디미트리]] '쥐지 않은 손' 대 다정함으로 흔드는 [[아드리안]] 집착. 19화에서 아드리안의 '매혹=가장 위험한 칼'임이 세실리아 내부에서 재확인됨. 감정 온도 분기 유지.

## 회수
- **[[황후]] 차회 호출(18화 열림)** → **19화 차회 개최로 진행·회수**: 호출의 실체가 '거절하는 손을 혼사로 묶는 하사'임이 드러남. 판은 닫히지 않고 혼사 압력으로 진화(→ 위 [중심]으로 승계).
- **17화 아드리안 적중의 정보 출처** → **19화 로잘린 질투발 실수로 소급 확정**(로잘린 채널이 '장부/이름'을 날림).
- **[[리타]] 보호 / '환한 자리'** → 18화 완결(유지·관리).
- **디미트리 화두 '성한 자리로 옮겨'(16화)** → 18화 결실(유지).
- **아드리안 전지전능 균열 재투입(18화)** → 유지(공개 공표 헛짚음).
- **세실리아 비밀 편지 봉쇄** → 15화 공개 자선 장부 전환 완결(유지).
- **'살아서 갚으라' 값([[디미트리]] 라인)** → 16화 일부 호출·거절, 이후 보류 유지.
- **변경 도피 봉쇄** → 16화 확정, 18화 '환한 자리' 실행으로 대체·완결.
- **카심 '모십니다' 어법 균열** → 15화 공개 우스개로 회수(유지).
- **운명의 무도회** → 9화 닫음(유지).
- **마르샹 자작가를 끌어 올린 손 / '벗삼아' 셋째 손** → 9화 폐하-황후 층위로 정리(유지).
