# 떡밥

> 생성기가 매 화 갱신([열림]→[회수]). 열린 떡밥이 소리없이 사라지면 연속성 체크가 잡는다.

## 열림
- [중심] **[[황후]]의 회귀 직격 — '다 끝나 본 눈'** — 23화 신설. 충정 서신 도발에 황후가 노여움 아닌 정확한 의심('처음 성소에 든 아이의 눈이 아니다')으로 응답. 회귀자 최대 금기가 황후 라인에 처음 노출. 수습 미정.
- [중심] **회귀의 비밀 = 신전·여신 라인** — 23화 입구 개통. 여신상 앞 처형 환각 봉인이 깨지고, 여사제 [[레아노라]]가 '여신께서 두 번 보신 얼굴'이라 의미심장하게 짚음. 회귀가 신전/여신과 연결될 장기 단서. 레아노라가 적인지 실마리인지 미정.
- [중심] **모함의 진짜 손 = [[황후]] (사냥개 아닌 주인)** — 투서에 황후 의전관 부봉. 23화 성소 대면에서 '의전관'을 두고 둘 다 아는 것을 둘 다 말하지 않는 줄타기로 봉합(세실리아 빚 없는 응수). 황후 라인 직접 공작 상존.
- [중심] **[[황후]]-[[아드리안]] 분리 쐐기** — '부수려는 황후' vs '가지려는 아드리안' 이해 분기 유지. 23화 부봉 출처를 [[로잘린]] 공으로 흡수시켜 세실리아 출처 은폐 봉합 성공. 균열 관리 지속.
- [중심] **[[발루아 백작]] 누명의 첫 씨앗 = '4황자 밀약' 소문** — 21화 충정 재프레이밍 1차 무력화, 22화 채널 역류. 발루아가 주인-개 사이 '먹이'로 끼인 위치 상존.
- [중심] **[[황후]]의 혼사 하사 — 시계가 폐하 손으로** — 식 시점·'그릇' 판단 공개 위임으로 무한 보류화. 혼사 자체는 닫히지 않음.
- [중심] **[[아드리안]]의 집착 — 로맨스 시동** — 23화 위협 아닌 다정(성소 곁자리, 손등 곁 한 뼘)으로 세실리아가 손을 못 빼는 첫 실점. 집착이 매혹의 결로 재가동. 주인을 향한 균열과 병행.
- [중심] **회피선 붕괴 = 2막 진입** — 세실리아는 공인 '공작가 약혼 후보'. 공개 대면이 상시 전장.
- [중심] **발루아의 위치 — '먹이'로 끼임** — 22화 폐하 직접 충정 서신에 착수, 23화 성소 대면으로 닿았으나 미결.
- [중심] **[[세실리아]]의 슬립 + 회귀 의심** — [[아드리안]] 보유 미해소 + 23화부터 [[황후]]·[[레아노라]] 라인에도 의심 신설.
- [중심] **[[로잘린]] = [[아드리안]] 직통 채널 (같은 둥지)** — 23화 부봉 진실을 위층에 나르고 그 공을 제 것으로 가로챔(세실리아 출처 은폐에 역설적 기여). 성소에서 행렬에도 황후 곁에도 못 끼고 빈손으로 손등만 매만지는 결핍 행동 비트(대사 없음). 채널 상존, 다음 함정 경계.
- [중심] **[[디미트리]] '쥐지 않은 손' — 비공개 통로** — 이번 화 직접 등장 없음. 온도 유지·진짜 인연 대비(아드리안 다정과 대조).
- [장기] **[[발루아 백작]] 지키기** — 곡물세 반려 유지+충신 프레임, 황후발 직접 공작+먹이 위치로 위험 최고조.
- [장기] **[[로잘린]]의 배신** — 동기(허기) 누적, 빈손으로 기다리는 결핍 심화.
- [장기] **회귀의 비밀** — 23화 신전·여신·[[레아노라]] 입구로 처음 굴러감.
- [암시] **변경의 칼 / [[디미트리]] 라인** — 비공개 통로 가동 유지.
- [열림] **[[엘비라 마르샹]] 우군화** — 부서-자작가 채널 상존(이번 화 직접 등장 없음).
- [열림] **[[발루아 백작 부인]](어머니)** — 곡물세 반려 가내 공유 미완(이번 화 직접 등장 없음).
- [열림] **세실리아 vs '진짜 인연' 대비** — 곁을 비우면서도 등을 봐주는 [[디미트리]] 대 다정함으로 가두려는 [[아드리안]]. 23화 성소 다정 비트로 대비 심화.

## 회수
- **22화 충정 도발 서신('나는 안다')** → 23화 황후의 침묵+성소 대면+회귀 직격으로 응답 도착(닫지 않고 [중심]으로 형질 전환·승계).
- **22화 부봉 출처 은폐 위험** → 23화 로잘린이 제 공으로 가로채 위층에 봉합([중심] 승계).
- **22화 분리 쐐기 실증** → 23화 출처 은폐로 균열 유지([중심] 승계).
- **22화 디미트리 '먹이' 경고** → 23화 충정 서신·성소 대면으로 '먹이 탈출' 시도 미결(승계).
- **회귀 의심 잠복(22화)** → 23화 황후·신전 라인으로 트리거 복원([중심] 승계).
- **발루아 무계파 상실(20화)→충신 복원(21화)→먹이 위치(22화)** → 23화 폐하 직접 충정으로 빼내기 시도(승계).
- **[[황후]] 혼사 하사 무한 보류(20화)** → 유지([중심] 승계).
- **[[리타]] 보호 / 공개 회계 공표** → 18화 완결(유지·관리).
- **운명의 무도회** → 9화 닫음(유지).
- **'벗삼아' 셋째 손** → 9화 폐하-황후 층위 정리(유지).
