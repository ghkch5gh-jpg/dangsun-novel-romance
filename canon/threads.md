# 떡밥

> 생성기가 매 화 갱신([열림]→[회수]). 열린 떡밥이 소리없이 사라지면 연속성 체크가 잡는다.

## 열림
- [중심] **[[아드리안]]의 집착 메커니즘 — 본인 직접 침투로 격상** — 12화에서 사물(봉서·초·쪽지)을 넘어 *본인이 정식 예방으로 응접실에 직접 들어옴*. 부모 앞 흠 없는 가면 / 단둘일 때 서늘함 양면. 가르쳐 준 적 없는 [[리타]] 이름을 부르며 '잠시 빌려 가겠다' 청 — 사적 영역으로의 침투가 사람(하녀)을 직접 가져가는 단계로 진입. 다음 수: 이 청에 대한 응전, 그리고 더 안쪽(세실리아 본인)으로의 접근.
- [중심] **무관심이라는 무기** — 12화 근접전에서 '초가 아까워서요'로 단둘 서늘함을 통제 응수. 단 본인이 집에 들어온 이상 무관심만으로 막을 수 없는 국면(리타 직격)에 도달.
- [중심] **[[세실리아]]의 슬립 한 조각** — 아드리안 보유 유지. 12화 회귀 자체는 미노출.
- [중심] **[[로잘린]] = [[아드리안]] 직통 채널 (같은 둥지)** — 12화 세실리아의 *역이용 1수* 실행(진실 같은 미끼: '두렵지 않다/하루 만에 기울었다'). 로잘린은 슬립 없이 어제 말과의 모순을 짚어 *되받아침*(위협 복원)했고, 세실리아는 그 짚음을 미끼의 신뢰도로 역전. 핀 출처 여전히 흐림. 다음: 미끼가 그자의 손을 실제로 늦추는지 확인.
- [중심] **[[황후]]의 판** — 빚 없이 거절한 발루아를 끌어들이려는 다음 손이 남음. 12화 직접 등장 없음(아침 후속 미도래, 다음 화 강력 후보).
- [장기] **[[발루아 백작]] 지키기** — 곡물세 반려 유지로 전생 죽음 통로 차단. 12화 등장 없음(딸의 위기에 다음 화 재진입 후보).
- [장기] **[[리타]] 지키기** — 12화 **최대 위기**. 아드리안이 부모 앞 정식 호의로 '잠시 빌려 가겠다' 청해 예법상 거절 명분이 막힘. 가르쳐 준 적 없는 이름까지 앎. 미해결로 다음 화로 넘어감.
- [장기] **[[로잘린]]의 배신** — 채널 성질 파악 후 역이용 단계 진입(1수 실행). 위협은 되받아침으로 복원.
- [장기] **회귀의 비밀** — 미해명 유지.
- [암시] **변경의 칼** — [[카심 다리브]]/[[디미트리]] 라인. '값은 살아서 갚으시라' 생존 신호 유지(12화 직접 등장 없음).
- [열림] **[[세실리아]]의 회귀 은폐** — 무관심·미끼·집 안 적 사이 줄타기에 더해, 본인이 집에 들어온 근접전에서의 은폐 압박 추가.
- [열림] **[[발루아 백작 부인]](어머니)의 진입** — 12화 진입. 묻지 않고 손을 데우는 온기 비트로 가내에 자리 잡음. 곡물세 반려 가내 공유는 미완.

## 회수
- **아드리안의 초+카드 / 세 번째 쪽지** → 11화 처리, 12화 '본인 직접 침투'로 전이·격상되어 열림 유지.
- **세실리아의 역이용 전략('진실인 척 빚은 것')** → 12화 1수 실행으로 착수(완결 아님, 효과 확인은 다음 화).
- **아드리안 매 인장 봉서** → 10화 개봉·회수. 11화 거짓 간파. 12화 소매 보관 유지.
- **운명의 무도회** → 9화에서 닫음(유지).
- **[[리타]]의 귀가 약속** → 9화 현관 재회로 회수(유지). 단 리타 보호는 12화 새 위기로 재점화.
- **[[디미트리]]와의 거래(곡물세 재협상)** → 9화 종결, 값 보류. '살아서 갚아라' 유효.
- **마르샹 자작가를 끌어 올린 손 / '벗삼아' 셋째 손** → 9화 폐하-황후 층위로 정리(유지).
