# 떡밥

> 생성기가 매 화 갱신([열림]→[회수]). 열린 떡밥이 소리없이 사라지면 연속성 체크가 잡는다.

## 열림
- [중심] **성소의 촛불 표식 = 회귀 물증** — 26화 봉인의 *사적 이전*. [[세실리아]]가 [[발드릭 대사제]]의 공개 읽기를 분리 쐐기 셈('부수려는 손/가지려는 손이 둘')으로 일단 묶었으나, 발드릭이 그 줄을 닫힌 문(황후 처소) 뒤로 가져가 분리 쐐기가 닿지 않는 사적 영역으로 이동. 봉인은 사람들 앞에서만 유효.
- [중심] **회귀의 비밀 = '부른 손'** — 26화 신변수·능동화. 성소 기록: '여신은 부름 없이 두 번 보지 않는다, 두 번째 눈길엔 부른 손이 있다.' 세실리아는 처형대에서 후회만 흘렸을 뿐 누구를 부른 적이 없는데 응답이 왔다 — 회귀를 부른 손의 정체가 핵심 추적 과제.
- [중심] **[[발드릭 대사제]] = 성소 표식의 황후 라인 통로 + 자기 읽기** — 26화 직접 등장. 황후 명으로 표식을 읽어 올리러 왔으나 세실리아 셈에 막혀 공개 읽기 보류, 줄은 닫힌 문으로 되가져감. '누가 너를 불렀느냐'를 스스로 캐는 눈 — 순수 황후 도구인지 제 셈을 가진 자인지 미정(세실리아가 못 읽은 변수).
- [중심] **[[리타]] 보호 균열 — 성소 축언 보류** — 26화 신설. 발드릭이 '신을 등진 손이 섞인 자선'이라며 발루아 구휼의 여신 축언을 보류, 18화 완결됐던 불가침 보호가 다시 들출 수 있는 상태로. 식은 지렛대가 신전·황후 라인으로 부활.
- [중심] **[[황후]]의 회귀 의심 — '죽어 본 눈'** — 26화 발드릭이 표식 줄을 처소로 되가져가 의심 물증 통로가 사적 영역에서 가동. 세실리아 셈 무효 구간.
- [중심] **모함의 진짜 손 = [[황후]] (사냥개 아닌 주인)** — 황후 라인 직접 공작 상존(이번 화 직접 진전 없음).
- [중심] **[[황후]]-[[아드리안]] 분리 쐐기** — 26화 발드릭을 두 손이 정반대로 당김('부수려는 황후' vs '가지려는 아드리안'). 세실리아가 그 구도를 발드릭 앞에서 직접 셈으로 써먹음. 균열 관리 지속.
- [중심] **[[아드리안]]의 집착 — 회귀 '앎' + 표식 지렛대** — 26화 '알면서도 못 막는' 상태 심화. 표식이 황후 처소(닫힌 문)로 가버려, 공작도 대사제급의 입을 함부로 못 누름. 안다고 다 쥐지 못하는 자리.
- [중심] **[[발루아 백작]] 누명의 첫 씨앗 = '4황자 밀약' 소문** — 24화 무력화, '먹이' 위치 구조는 미결.
- [중심] **[[황후]]의 혼사 하사 — 시계가 폐하 손으로** — 무한 보류화 유지. 혼사 자체는 닫히지 않음.
- [중심] **회피선 붕괴 = 2막 진입** — 세실리아는 공인 '공작가 약혼 후보'. 공개 대면 상시 전장.
- [중심] **발루아의 위치 — '먹이'로 끼임** — 주인-개 사이 구조적 위치 잔존(이번 화 직접 진전 없음).
- [중심] **[[세실리아]]의 슬립 + 회귀 '앎'** — [[아드리안]] 보유(표식 물증+자백) + [[황후]]·[[발드릭]]·[[레아노라]] 라인 의심 잔존.
- [중심] **[[로잘린]] = 끊긴 채널의 반동** — 26화 행동 전환. 아드리안 함구령으로 위층 문이 닫히자, 황후 시녀 라인에 소곤이며 새 주인을 모색하는 능동 행보 개시.
- [중심] **[[디미트리]] '쥐지 않은 손' — 진짜 인연 대비** — 26화 카심 쪽지로 '발드릭 황후 처소 재입성' 정보 전달. 어둠 속 등 대주기 유지.
- [열림] **[[레아노라]] = 적/아군 미정** — 26화 백 년 묵은 기록('문턱을 두 번 건넌 얼굴', '부른 손')을 외워 협조하나, '두려우냐'는 읽히지 않는 눈. 보호적이되 속내 미정.
- [장기] **[[발루아 백작]] 지키기** — 황후발 직접 공작+먹이 위치+자선 축언 균열로 위험 지속.
- [장기] **[[로잘린]]의 배신** — 동기(허기·결핍) 누적, 채널 차단의 반동이 새 주인 모색으로 가시화.
- [장기] **회귀의 비밀** — '부른 손'·신전·여신·[[레아노라]]·[[발드릭 대사제]]·성소 표식 라인으로 능동 추적.
- [암시] **변경의 칼 / [[디미트리]] 라인** — 비공개 통로 가동 유지(카심 쪽지).
- [열림] **[[엘비라 마르샹]] 우군화** — 부서-자작가 채널 상존(이번 화 직접 등장 없음).
- [열림] **[[발루아 백작 부인]](어머니)** — 곡물세 반려 가내 공유 미완(이번 화 직접 등장 없음).
- [열림] **세실리아 vs '진짜 인연' 대비** — 26화 디미트리의 '못 막는 선을 정직히 긋는 손'(카심 쪽지) 유지.

## 회수
- **25화 '여신이 무엇을 두 번 보았는지 먼저 캐기' 결심** → 26화 성소 기록에서 '문턱을 두 번 건넌 얼굴'·'부른 손' 줄로 첫 단서 회수, 추적 능동화([중심] 승계).
- **24~25화 '그 줄을 외우는 다른 눈' = [[발드릭 대사제]]** → 26화 직접 등장으로 실체화, 황후 처소 출입 결과(표식 읽기 명·축언 보류) 가시화([중심] 승계).
- **25화 아드리안 손의 표식 봉인(조건부)** → 26화 분리 쐐기로 공개 읽기 일단 묶음, 단 닫힌 문으로 이전되어 한계 노출([중심] 승계).
- **25화 로잘린 채널 일시 차단** → 26화 반동이 황후 시녀 라인 새 주인 모색 행동으로 전환([중심] 승계).
- **18화 [[리타]] 보호 완결(불가침)** → 26화 성소 축언 보류로 균열, 지렛대 재가동(재열림).
- **24~25화 디미트리 비공개 통로** → 26화 카심 쪽지로 발드릭 재입성 정보 전달(승계).
- **[[황후]] 혼사 하사 무한 보류** → 유지([중심] 승계).
- **운명의 무도회** → 9화 닫음(유지). **'벗삼아' 셋째 손** → 9화 정리(유지).
