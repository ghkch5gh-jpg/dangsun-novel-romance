# 떡밥

> 생성기가 매 화 갱신([열림]→[회수]). 열린 떡밥이 소리없이 사라지면 연속성 체크가 잡는다.

## 열림
- [중심] **모함의 진짜 손 = [[황후]] (사냥개 아닌 주인)** — 투서에 황후 의전관 부봉. 22화에서 세실리아가 이 정보 우위를 *능동 사용*: 채널로 아드리안에게 역류시키고, 황후에게 '의전관' 두 글자를 박은 충정 서신으로 '나는 안다'를 도발. 황후 라인의 응답 미정.
- [중심] **[[황후]]-[[아드리안]] 같은 그늘 라인 — 분리 쐐기 실증** — 22화 본문 확정: '부수려는 황후' vs '가지려는 아드리안'으로 이해 분기. 아드리안이 처음 주인에게 적의('폐하의 사람이라 해도'). 둘을 가를 틈 능동 확대 중.
- [중심] **[[발루아 백작]] 누명의 첫 씨앗 = '4황자 밀약' 소문** — 21화 충정 재프레이밍으로 1차 무력화. 단 발루아가 주인-개 싸움의 '먹이'로 끼인 위치(디미트리 경고)라 위기 상존.
- [중심] **[[황후]]의 혼사 하사 — 시계가 폐하 손으로** — 식 시점·'그릇' 판단 공개 위임으로 무한 보류화. 혼사 자체는 닫히지 않음. 발루아를 그늘에 묶는 수와 병행.
- [중심] **[[아드리안]]의 집착 — 주인에게 돌아선 사냥개** — 보호 미끼 거절(21화) 뒤, 22화에서 제 위의 손(황후 라인)이 발루아를 친 걸 알고 처음으로 주인을 향해 이빨. '제 것에 손대는 건 봐줄 수 없다'. 다음 수 미정.
- [중심] **회피선 붕괴 = 2막 진입** — 세실리아는 공인 '공작가 약혼 후보'. 공개 대면이 상시 전장.
- [중심] **발루아의 위치 — '먹이'로 끼임** — 주인(황후)이 꺾으려 하고 개(아드리안)가 삼키려 하는 사이에 낀 뼈다귀. 22화 세실리아가 '먹이이기를 그만두는' 수(폐하 직접 충정 서신)에 착수.
- [중심] **[[세실리아]]의 슬립 + 회귀 의심** — [[아드리안]] 보유, 미해소. 22화 잠복.
- [중심] **[[로잘린]] = [[아드리안]] 직통 채널 (같은 둥지)** — 22화 처음으로 세실리아에게 *역이용*당함(부봉 진실을 위층에 나름). 쓸모 인정받고픈 허기로 '전해 주겠다' 자청·편지지 묶음을 능동적으로 당김(동기=결핍, 대사 없음). 채널은 여전히 상존, 다음 함정 경계. 핀 출처 흐림 유지.
- [중심] **[[디미트리]] '쥐지 않은 손' — 비공개 통로** — 22화 카심 쪽지로 '발루아가 둘 사이 먹이로 먼저 바스러진다' 경고+'내려선 자리에서 등은 봐준다' 온도. 물러선 자리의 온기가 가까운 어느 손보다 따뜻함 — 진짜 인연 대비 심화.
- [중심] **[[세실리아]]의 자발적 도박 — 황후에게 '나는 안다'** — 회귀자 금기(아는 척)를 스스로 넘어 정보 우위를 능동 사용. 들키면 가장 먼저 죽는 패를 먼저 내보임. 결과 미정.
- [장기] **[[발루아 백작]] 지키기** — 곡물세 반려 유지+충신 프레임, 그러나 황후발 직접 공작+먹이 위치로 위험 최고조.
- [장기] **[[로잘린]]의 배신** — 동기(허기) 누적, 역이용당한 뒤 다음 함정 대기.
- [장기] **회귀의 비밀** — 미해명 유지.
- [암시] **변경의 칼 / [[디미트리]] 라인** — 비공개 통로 가동 유지.
- [열림] **[[엘비라 마르샹]] 우군화** — 부서-자작가 채널 상존(이번 화 직접 등장 없음).
- [열림] **[[세실리아]]의 회귀 은폐** — 의심 상존.
- [열림] **[[발루아 백작 부인]](어머니)** — 곡물세 반려 가내 공유 미완(이번 화 직접 등장 없음).
- [열림] **세실리아 vs '진짜 인연' 대비** — 곁을 비우면서도 어둠 속 등을 봐주는 [[디미트리]] 대 다정함으로 가두려는 [[아드리안]]. 22화 카심 쪽지 온도로 진전.

## 회수
- **'21화 부봉=황후 손' 정보 우위** → 22화 채널 역류+도발 서신으로 능동 사용(닫지 않고 [중심] 승계, 굴림).
- **아드리안 보호 미끼 반격(21화)** → 22화 주인에게 돌아선 사냥개로 형질 전환([중심] 승계).
- **아드리안 시간표 박탈(20화)** → 21화 보호 미끼→차단, 22화 주인 향한 적의로 전환(승계).
- **발루아 무계파 상실(20화)** → 21화 충신 재프레이밍 반걸음 복원, 22화 '먹이 위치'로 위험 재격상(승계).
- **디미트리 공개 후퇴(20화)** → 21~22화 비공개 통로로 온도 유지(승계).
- **[[황후]] 혼사 하사 '그릇' 보류(19화)→공개 위임(20화)** → 무한 보류 유지([중심] 승계).
- **17화 아드리안 적중의 정보 출처** → 19화 로잘린 질투발 실수로 소급 확정(유지).
- **[[리타]] 보호 / 공개 회계 공표** → 18화 완결(유지·관리).
- **세실리아 비밀 편지 봉쇄** → 15화 공개 자선 전환 완결(유지).
- **'살아서 갚으라' 값([[디미트리]] 라인)** → 16화 일부 호출·거절, 보류 유지.
- **운명의 무도회** → 9화 닫음(유지).
- **'벗삼아' 셋째 손** → 9화 폐하-황후 층위 정리(유지).
