# 떡밥

> 생성기가 매 화 갱신([열림]→[회수]). 열린 떡밥이 소리없이 사라지면 연속성 체크가 잡는다.

## 열림
- [중심] **성소의 촛불 표식 = 회귀 물증** — 24화 신설. 바람 없이 불꽃이 사람을 향해 누우면 '여신이 두 번 보신 표식'(백 년 만의 기록). [[레아노라]]가 끄며 가렸으나 **[[아드리안]]이 목격해 절단 대사로 제 입에 올림** — 회귀 비밀이 처음으로 사람 입이 아닌 물증으로 그자 손에. 최우선 위기.
- [중심] **회귀의 비밀 = 신전·여신 라인** — 24화 전진. 촛불 표식·백 년 만의 기록, [[레아노라]]의 '초를 멀리 두라' 경고, '그 줄을 외우는 눈이 저 하나만 있지 않다'(신전 내 다른 감지자) 단서. 레아노라 적/실마리 여전히 미정이나 보호적 정황↑.
- [중심] **[[황후]]의 회귀 의심 — '죽어 본 눈'** — 24화 직격에 [[세실리아]]가 '열병' 변명으로 둘러댔으나 황후가 '앓은 눈과 죽어 본 눈은 다르다'고 받아쳐 **회귀 후 처음으로 완전히 받아치지 못하고 한 발 밀림**. 의심 잔존·미봉합.
- [중심] **모함의 진짜 손 = [[황후]] (사냥개 아닌 주인)** — 투서에 황후 의전관 부봉. 황후 라인 직접 공작 상존.
- [중심] **[[황후]]-[[아드리안]] 분리 쐐기** — '부수려는 황후' vs '가지려는 아드리안' 이해 분기 유지. 균열 관리 지속.
- [중심] **[[발루아 백작]] 누명의 첫 씨앗 = '4황자 밀약' 소문** — 21화 1차 무력화 후 **24화 [[아드리안]]이 [[카심 다리브]] 4황자 신분 폭로로 재점화**. 세실리아의 공개 장부 논리+[[디미트리]] 공개 방어로 '자선'으로 뒤집어 무력화. 발루아 '먹이' 위치 구조는 미결.
- [중심] **[[황후]]의 혼사 하사 — 시계가 폐하 손으로** — 무한 보류화 유지. 혼사 자체는 닫히지 않음.
- [중심] **[[아드리안]]의 집착 — 로맨스 시동 + 행동 수** — 24화 봉서·다정이 아닌 *공개 폭로*라는 판 흔드는 행동 수로 진화(언어 집착 정체 해소). 마지막에 성소 표식을 쥐고 회귀 의심을 직접 압박.
- [중심] **회피선 붕괴 = 2막 진입** — 세실리아는 공인 '공작가 약혼 후보'. 공개 대면이 상시 전장.
- [중심] **발루아의 위치 — '먹이'로 끼임** — 24화 밀약 칼은 막았으나 주인-개 사이 구조적 위치 잔존.
- [중심] **[[세실리아]]의 슬립 + 회귀 의심** — [[아드리안]] 보유(+성소 표식 물증) + [[황후]]·[[레아노라]] 라인 의심 신설·잔존.
- [중심] **[[로잘린]] = [[아드리안]] 직통 채널 (같은 둥지)** — 24화 성소 대화를 능동 탐문('무슨 말씀 나눴느냐')해 위층에 나르려 하나 세실리아의 '불씨' 미끼에 가져갈 곳을 잃음. 채널·감시 상존, 다음 함정 경계.
- [중심] **[[디미트리]] '쥐지 않은 손' — 진짜 인연 대비** — 24화 공개 동맹 후퇴를 깨고 알현 회랑에 직접 재등장, 제 발로 칼 앞에 서 발루아를 방어하되 제 편이라 칭하지 않음(쥐지 않은 손). 본인 재노출 대가 발생. 비공개 통로 병행.
- [장기] **[[발루아 백작]] 지키기** — 황후발 직접 공작+먹이 위치로 위험 최고조.
- [장기] **[[로잘린]]의 배신** — 동기(허기·결핍) 누적, 능동 탐문으로 전환.
- [장기] **회귀의 비밀** — 신전·여신·[[레아노라]]·성소 표식 라인으로 본격 전진.
- [암시] **변경의 칼 / [[디미트리]] 라인** — 24화 카심 신분 공개 노출, 비공개 통로 가동 유지.
- [열림] **[[엘비라 마르샹]] 우군화** — 부서-자작가 채널 상존(이번 화 직접 등장 없음).
- [열림] **[[발루아 백작 부인]](어머니)** — 곡물세 반려 가내 공유 미완(이번 화 직접 등장 없음).
- [열림] **세실리아 vs '진짜 인연' 대비** — 24화 디미트리가 제 발로 칼 앞에 서는 행동으로 대비 심화(가두려는 아드리안 대 묶지 않는 디미트리).

## 회수
- **23화 황후 '다 끝나 본 눈' 직격** → 24화 세실리아가 '열병'으로 둘러댔으나 완전 봉합 실패·한 발 밀림([중심] 승계).
- **23화 신전·여신 입구(레아노라)** → 24화 촛불 표식·백 년 기록·'다른 눈' 단서로 전진([중심] 승계).
- **21~22화 '4황자 밀약' 소문** → 24화 아드리안의 카심 신분 공개 폭로로 재점화, 자선 논리+디미트리 공개 방어로 무력화(승계·미결).
- **20화 디미트리 공개 동맹 후퇴** → 24화 직접 재등장으로 일부 회수(본인 재노출 대가, [중심] 승계).
- **23화 부봉 출처 은폐(로잘린 공 가로채기)** → 24화 채널 상존·탐문 능동화로 승계.
- **22화 충정 도발 서신** → 23화 성소 대면 응답, 24화 직격 수습 미완으로 승계.
- **[[황후]] 혼사 하사 무한 보류** → 유지([중심] 승계).
- **[[리타]] 보호 / 공개 회계 공표** → 18화 완결(유지·관리).
- **운명의 무도회** → 9화 닫음(유지). **'벗삼아' 셋째 손** → 9화 정리(유지).
