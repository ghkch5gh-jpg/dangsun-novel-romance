# 떡밥

> 생성기가 매 화 갱신([열림]→[회수]). 열린 떡밥이 소리없이 사라지면 연속성 체크가 잡는다.

## 열림
- [중심] **[[아드리안]]의 집착 — 약점 확보 단계로 격상** — 본인 직접 침투(정식 예방)에 이어, 13화에서 [[리타]] 빌리기 청이 '무엇을 가장 아끼는지' 재 보려는 *미끼*였음이 드러남. 리타=세실리아 최대 약점을 손에 넣고 떠남('어디를 두드리면 흔들리는지 알겠다'). 다음 수: 리타를 지렛대로 한 본인 접근.
- [중심] **무관심이라는 무기** — '초가 아까워서요'로 단둘 서늘함 통제는 유지. 단 약점(리타)을 들킨 이상 무관심만으로 막을 수 없는 국면 심화.
- [중심] **[[세실리아]]의 슬립 한 조각** — 아드리안 보유 유지. 회귀 자체는 미노출.
- [중심] **[[로잘린]] = [[아드리안]] 직통 채널 (같은 둥지)** — 13화 리타 이름이 위층으로 간 통로로 세실리아가 채널을 재확인(아드리안은 전지 아님 — 채널이 부지런했을 뿐). 마지막에 세실리아의 *비밀 편지 현장*에 등장해 채널이 세실리아의 반격까지 겨눔. 핀 출처 흐림 유지.
- [중심] **[[황후]]의 판** — 빚 없이 빠진 발루아를 끌어들일 다음 손 남음. 13화 직접 등장 없음(아침 후속 미도래, 다음 화 후보).
- [장기] **[[발루아 백작]] 지키기** — 곡물세 반려 유지로 전생 죽음 통로 차단. 13화 등장 없음(딸의 능동책에 다음 화 재진입 후보).
- [장기] **[[리타]] 지키기** — 13화 1차 위기는 어머니 판으로 해소(빌려 가기 무산), 그러나 표적 확정. 세실리아가 집 밖(디미트리 라인)으로 빼내는 *능동책 착수*(미발송).
- [장기] **[[로잘린]]의 배신** — 채널 역이용 지속 + 이제 세실리아의 반격을 감시하는 위치로 전환.
- [장기] **회귀의 비밀** — 미해명 유지.
- [암시] **변경의 칼** — [[카심 다리브]]/[[디미트리]] 라인. 세실리아가 '살아서 갚아라' 값을 *처음으로* 쓰며 우군화 착수.
- [열림] **[[세실리아]]의 회귀 은폐** — 집 안 채널 앞 비밀 편지 은폐라는 새 압박 추가.
- [열림] **[[발루아 백작 부인]](어머니)** — '문 여는 손'에서 '안주인의 판으로 문 닫는 손'으로 전환, 우군 자원으로 가시화. 곡물세 반려 가내 공유는 여전히 미완.
- [열림/신규] **세실리아의 첫 능동책(집 밖에 손 두기)** — 디미트리 라인행 비밀 편지. 발송 성패·로잘린 감시 돌파가 다음 화 핵.

## 회수
- **리타 '잠시 빌려 가겠다' 위기** → 13화 어머니 판으로 1차 해소(빌려 가기 무산). 단 '약점 확보'로 전이되어 열림 유지.
- **아드리안의 리타 이름 출처** → 13화 [[로잘린]] 채널로 납득·회수(전지 아님, 채널 설정과 정합).
- **아드리안의 초+카드 / 세 번째 쪽지** → 11화 처리, 12~13화 '본인 직접 침투'로 전이.
- **세실리아의 역이용 전략('진실인 척 빚은 것')** → 12화 1수, 13화 미끼 효과는 아직 미확인(아드리안이 도리어 약점을 봄 — 부분 역효과).
- **아드리안 매 인장 봉서** → 10화 개봉·회수. 13화 소매 보관 유지.
- **운명의 무도회** → 9화에서 닫음(유지).
- **[[리타]]의 귀가 약속** → 9화 현관 재회로 회수. 13화 재위기 1차 해소.
- **[[디미트리]]와의 거래(곡물세 재협상)** → 9화 종결, 값 보류. '살아서 갚아라'를 13화에 처음 집행 착수.
- **마르샹 자작가를 끌어 올린 손 / '벗삼아' 셋째 손** → 9화 폐하-황후 층위로 정리(유지).
