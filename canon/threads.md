# 떡밥

> 생성기가 매 화 갱신([열림]→[회수]). 열린 떡밥이 소리없이 사라지면 연속성 체크가 잡는다.

## 열림
- [중심] **[[아드리안]]의 집착 — 회귀 의심(전지전능 균열 재발)** — 17화 정확 적중 직후, 18화엔 '리타를 어둠으로 빼돌릴 것'이라 걸고 발루아 뒷문을 야간 감시했으나 *공개 공표*를 예측 못 해 헛짚음 — 전지전능에 균열 재투입. 회귀 의심 자체는 미해소. 다음 수: 공개 공표를 어떻게 재해석/공략하는가.
- [중심] **무관심이라는 무기** — 답장 0 유지. 아버지 앞 봉서에도 직접 응대하지 않음.
- [중심] **[[세실리아]]의 슬립 한 조각 + 회귀 의심** — [[아드리안]] 보유, 미해소. 18화 균열로 압박 일시 둔화.
- [중심] **[[로잘린]] = [[아드리안]] 직통 채널 (같은 둥지)** — 집 안 상존. 18화엔 슬립 없이 가면 밑 '제 편 가져 본 적 없는 자'의 허기 후속('난 그런 거 가져 본 적이 없어서') → 배신 동기(결핍·소외) 누적. 단 이번 수는 '이미 공개될 이름'이라 채널이 위층에 날라도 무해(함정 무력화). 핀 출처 흐림 유지.
- [중심] **[[황후]]의 판** — 18화 차회 직접 호출(의전관 부봉 봉서)로 재가동. '거절할 줄 아는 손을 가까이서 보고 싶다'며 세실리아를 직접 자기 자리로 부름. 막경계(20화) 강력 후보.
- [장기] **[[발루아 백작]] 지키기** — 곡물세 반려 유지. 18화 당주 인장으로 리타 공표에 동참, 막내딸의 성장을 인정.
- [장기] **[[리타]] 지키기 / '환한 자리' 보호** — **18화 완결**(→회수). 공표·봉인 이후 유지·관리만 남음.
- [장기] **[[로잘린]]의 배신** — 동기 단서(허기) 누적. 다음 함정 예고.
- [장기] **회귀의 비밀** — 미해명 유지.
- [암시] **변경의 칼 / [[디미트리]] 라인 우군화** — 18화 직접 재등장(장부실). 부서 입회로 리타 공표를 성사시켜 화두의 결실에 동석, 곡물세/황후 정치 라인 재가동. '쥐지 않은 손' 온도(진짜 인연 대비)는 보류. 변경의 칼 정체 미확정.
- [열림] **[[엘비라 마르샹]] 우군화** — 18화 직접 등장 없음. 부서-자작가 결합 채널 가동 중.
- [열림] **[[세실리아]]의 회귀 은폐** — 18화 아드리안 균열로 압박 일시 완화, 단 의심 상존.
- [열림] **[[발루아 백작 부인]](어머니)** — 안주인 인장으로 리타 공표에 동참(관문 통과). 곡물세 반려 가내 공유는 여전히 미완.
- [열림] **세실리아 vs '진짜 인연' 대비** — 약점을 알고도 안 쑤시는 [[디미트리]] '쥐지 않은 손' 대 약점을 정확히 쑤시는 [[아드리안]] 집착. 감정 온도 분기 유지.

## 회수
- **[[리타]] 보호 / '환한 자리'** → **18화 완결**: 당주·안주인 인장+부서 입회·봉인으로 발루아 자선 장부 첫 줄 회계 담당 공표. 6화째 끌던 보호건 종결(회귀 후 두 번째 완결 1승).
- **디미트리 화두 '성한 자리로 옮겨'(16화)** → 18화 부서 입회로 완전 결실(리타 공표 성사).
- **아드리안 전지전능 복원(17화)** → 18화 야간 빼돌리기 오판으로 균열 재투입.
- **아드리안 '디미트리 구애' 질투 오독(15~16화)** → 17화 종료(유지).
- **세실리아 첫 능동책 / 비밀 편지 봉쇄** → 15화 공개 자선 장부 전환으로 완결(유지).
- **'살아서 갚으라' 값 ([[디미트리]] 라인)** → 16화 일부 호출·거절. 17·18화 미호출(보류 유지).
- **변경 도피 봉쇄** → 16화 완전 확정. 18화 '환한 자리에 세우기' 실행으로 보호 방향 대체·완결.
- **카심 '모십니다' 어법 균열** → 15화 공개 우스개로 회수(유지).
- **리타 '잠시 빌려 가겠다' 위기** → 13화 1차 해소·약점 전이 → 18화 보호 완결로 응전 종결.
- **운명의 무도회** → 9화 닫음(유지).
- **마르샹 자작가를 끌어 올린 손 / '벗삼아' 셋째 손** → 9화 폐하-황후 층위로 정리(유지).
