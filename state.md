# 현재 상태

> 런닝 — 매 화 끝 시점의 스냅샷으로 갱신.

- **회차**: 18화 종료 시점
- **장소**: 발루아 저택 서재 → 어머니 방/복도 → (다음 날) 황궁 사교부 변경 구휼 장부실 → 발루아 저택 현관.
- **시각**: 봄, 17화 오후에서 하루 경과. 아침(서재·장부실) → 낮 귀가.
- **소지**: 전생의 기억 전부. 곡물세 반려 유지(전생 아버지 죽음 통로 차단). [[황후]]에게 빚 없이 디딘 한 걸음. [[디미트리]]가 끊어 준 라일락 가지. 변경 구휼 공개 장부(발루아+마르샹, 곡물 부서 입회). [[엘비라 마르샹]] 우군 겸 공개 통로. **[[아드리안]]의 매 인장 봉서 — 세실리아 앞 세 장(소매 보관) + 아버지 앞 한 장. 무관심·답장 0 유지.** **[[리타]] 보호 완결 — 발루아 자선 장부 첫 줄 회계 담당으로 당주·안주인 인장+부서 입회·봉인. 불가침 성립.** **새로: [[황후]] 차회 호출 봉서(의전관 부봉).**
- **신체**: 위기·온기에 목멤 1회(어머니)로만 표출, 그 외 절제. 처형 환각·밀랍 압인 묘사 0회(봉인 유지).
- **즉시 목표**: (1) **[[리타]] 보호 — 완결**(공표·봉인). 유지·관리 단계로 전환. (2) **[[황후]] 차회** — 능동 수락. 판을 먼저 읽고 들어가 빚 없이 빠지기(막경계 입구). (3) **[[아드리안]] 회귀 의심** — 야간 빼돌리기 오판으로 균열 재발생, 단 의심 자체는 미해소(공개 공표를 어떻게 재해석할지 미지). (4) **[[로잘린]] 채널·동기** — 허기 후속 누적, 채널 상존하나 '공개될 이름'엔 함정 무력. (5) **[[디미트리]] 라인 온도·곡물세 판 재요동.**
- **장기 목표**: [[발루아 백작]] 살리기(통로 차단 유지) / [[리타]] 지키기(보호 완결, 유지) / [[로잘린]] 배신 차단(동기=결핍·허기 누적) / [[아드리안]] 집착·회귀 의심 차단(균열 1회, 의심 상존) / 회귀의 비밀 추적 / [[황후]]-[[아드리안]] 라인 해체 / [[발루아 백작 부인]] 관계 / [[디미트리]]·[[엘비라 마르샹]] 우군화 심화.
- **들키지 말 것**: 회귀 사실([[아드리안]] 의심 상존, 단 공개 공표는 헛짚음으로 일시 둔화). 진짜 목적. 리타=약점은 13·17화에 들켰으나 18화에 *불가침*으로 전환. [[로잘린]] 채널 상존. [[디미트리]]엔 '데어 봤다' 시인 유지.
- **새 변수**: (1) **[[리타]] 보호 완결** — 자력 해법 실행 성공(회귀 후 두 번째 완결 1승, 6화째 끌던 보호건 종결). (2) **[[아드리안]] 전지전능 균열 재투입** — 17화 정확 적중 직후, 18화엔 '야간에 빼돌릴 것'이라 걸고 뒷문 감시했으나 공개 공표를 예측 못 해 헛짚음. (3) **[[황후]] 판 재가동** — 차회 직접 호출(의전관 경유), 9화 이후 식은 정치 라인 재점화·20화 막경계 도약 입구. (4) **[[디미트리]] 직접 재등장** — 부서 입회로 리타 공표 성사 동석, 곡물세 반려 유지 확인+황후 차회 예고.

## 다음 화 방향
18화 끝, [[황후]] 차회 호출을 능동 수락한 지점에서 이어감(1막 종반, 20화 막경계 도약). (1) **황후 차회 진입** — '거절할 줄 아는 손' 흥미의 실체, 황후 판의 다음 손, 빚 없이 빠져나오는 줄타기. (2) **[[아드리안]]** — 공개 공표를 어떻게 재해석/재공략하는가(균열 뒤 다음 수). (3) **[[로잘린]] 동기** — '허기' 후속 한 스푼(본색 노출 전). (4) **[[디미트리]] 곡물세 판** — 황후가 흔드는 판의 후속. 페이싱: 황후 차회로 정치 국면을 끌어올려 20화 막경계 도약 준비.
