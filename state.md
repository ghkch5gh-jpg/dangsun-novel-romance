# 현재 상태

> 런닝 — 매 화 끝 시점의 스냅샷으로 갱신.

- **회차**: 12화 종료 시점
- **장소**: 발루아 백작 저택 응접실.
- **시각**: 봄, 무도회 이틀 뒤. 아침 ~ 오후 세 시.
- **소지**: 전생의 기억 전부. 곡물세 반려 유지(전생 아버지 죽음 통로 차단). [[황후]]에게 빚 없이 디딘 한 걸음. [[디미트리]]가 끊어 준 라일락 가지. **[[아드리안]]의 매 인장 봉서(소매 보관, '태웠다'는 거짓은 이미 간파됨).** [[디미트리]]의 보류된 값 '살아서 갚아라' 유효. 아드리안이 보낸 초(부엌 장작)·세 번째 쪽지(태움)는 처리 완료.
- **신체**: 어머니 손에 손을 맡긴 채 말이 목에 막혀 이마를 어깨에 댐(트라우마는 망설임·말막힘으로만). 리타를 빌려 달라는 청 앞에서 막아설 말을 한 줄도 못 찾는 정지 1회.
- **즉시 목표**: (1) **머리로 못 푸는 위기 발생 — [[리타]] 직격**. 아드리안이 부모 앞 정식 호의의 형식으로 리타를 '잠시 빌려 가겠다' 청함. 예법상 거절=모욕이라 명분이 없음. 이번 화 내 미해결. (2) **로잘린 채널 역이용 1수 실행** — '두렵지 않다/하루 만에 기울었다'는 진실 같은 미끼를 얹어 그자의 손을 늦추려는 설계. 단 로잘린이 어제 말과의 모순을 짚어 *되받아침*(위협 복원), 세실리아는 그 짚음을 미끼의 신뢰도로 역전. (3) **무관심 유지** — 단둘 서늘함에 '초가 아까워서요'로 통제 응수. (4) [[황후]] 판 아침 후속 미도래(여전히 열림). (5) [[디미트리]] 값 청구 시점 유효.
- **장기 목표**: [[발루아 백작]] 살리기(통로 차단 유지) / [[리타]] 지키기(**최대 위기 — 아드리안이 직접 빼가려 함**) / [[로잘린]] 배신 차단(역이용 1수 실행, 위협 복원) / [[아드리안]] 집착의 가내 침투 차단(본인 침투로 격상) / 회귀의 비밀 추적 / 황후-아드리안 라인 해체 / [[발루아 백작 부인]]과의 관계 지키기
- **들키지 말 것**: 회귀 사실. [[아드리안]]은 회피의 슬립 조각 보유 + 집 안 줄로 거짓 간파 + 이제 본인이 집에 들어와 직접 압박. [[로잘린]] 채널이 '진실만 정확히 나름' 입증됨. [[디미트리]]엔 '기억 진짜·한계 있다' 노선. [[황후]] 앞 가면 대 가면.
- **새 변수**: (1) **아드리안 본인 직접 방문** — 사적 수가 사물(초·쪽지)에서 *본인 침투*로 격상. 부모 앞 흠 없는 가면 / 단둘일 때 서늘함 양면. (2) **아드리안이 리타의 이름을 앎** — 가르쳐 준 적 없는 이름을 불러 가내 침투의 깊이를 증명. '빌려 가겠다' 청으로 리타 직격. (3) **세실리아의 진실 같은 미끼 1수 가동** — 로잘린이 모순으로 되받아치자 변덕을 미끼의 살로 역전. (4) **[[발루아 백작 부인]] 진입** — 묻지 않고 손을 데우는 온기 비트.

## 다음 화 방향
12화 끝, 아드리안이 부모 앞에서 [[리타]]를 '잠시 빌려 가고 싶다'고 정식으로 청하고 가르쳐 준 적 없는 이름을 부른 직후에서 이어감(1막 후반, 머리로 못 푸는 위기 한복판). (1) **리타 위기 해소** — 예법의 칼을 어떻게 빗겨 거절 명분을 만들 것인가. 전생 지식·우군(아버지/디미트리 라인)을 동원하되 회귀를 들키지 않는 줄타기. (2) **아드리안 본인 대 세실리아 근접전** — 집 안에 들어온 그자를 어떻게 물릴 것인가, 미끼의 효과(손을 늦추는지) 확인. (3) **황후 판 아침 후속** — 다음 화 진입 강력 후보 유지. (4) [[발루아 백작]] 본격 재진입(딸의 위기를 아비가 어떻게 받치는가). (5) [[디미트리]] 값 청구. 페이싱: '집 안에 들어온 집착'과 '리타 사수'를 축으로 1막 후반 절정, 20화 막경계 도약 준비.
